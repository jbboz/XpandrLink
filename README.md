# XpandrLink

**A real-time editor, librarian, and DAW plugin for the Oberheim Xpander and Matrix-12.**

Standalone app, Audio Unit, and VST3 for macOS (Apple Silicon + Intel) and Windows.

Full documentation, feature list, and build instructions land later in this commit
series as the source and docs are added — see the project history for the complete
picture, or jump to the final commit for the finished README.
