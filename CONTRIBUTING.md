# Contributing

Thanks for your interest in XpandrLink.

## Reporting Issues

Open a GitHub issue with your platform (macOS/Windows), synth (Xpander/Matrix-12), and host (Standalone/AU/VST3 + DAW if applicable). For MIDI/protocol bugs, a description of what was sent/received (visible in the app's log panel) is very helpful.

## Pull Requests

PRs are welcome. Keep changes focused and consistent with the existing code style (see [SPEC.md](SPEC.md) for architecture and implementation invariants). Hardware protocol changes should be cross-checked against the [upstream XplorerEditor source](https://github.com/xplorer2716/XplorerEditor) where possible — see [SPEC.md §9](SPEC.md#9-references) for a file-by-file index.
