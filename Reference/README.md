# Reference

`OberheimXpanderMidiSpec.pdf` / `.doc` — the Oberheim Xpander/Matrix-12 MIDI specification
this project's SysEx protocol implementation is built against.

**Source and license**: this document is maintained by the author of the upstream project
[XplorerEditor](https://github.com/xplorer2716/XplorerEditor) this project is a C++/JUCE port
of, in their own repo:
[github.com/xplorer2716/OberheimXpanderMidiSpec](https://github.com/xplorer2716/OberheimXpanderMidiSpec).
Per that repo's README, it's a rework of three older plain-text community documents (not a
scan of Oberheim's own printed manual), released under **GPL-3.0** — the same license as
this project and as `XplorerEditor`.

Vendored here (rather than only linked) so it stays available for archival/retrieval purposes
even if the original community sources it was reworked from — or any single mirror of it —
go offline, as has already happened to some of the pages it cites. Check the upstream repo
for the latest version.
